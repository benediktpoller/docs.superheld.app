# Nutzung

Nach der Installation starten Sie die App und richten ein Benutzerkonto ein oder melden sich mit einem bestehenden an. 

Das Hauptziel ist es, Ihre Datenflüsse zu kontrollieren und zu anonymisieren. Die Oberfläche ist für normale Nutzer bewusst einfach gehalten:

1. **Sprach- oder Texteingabe**: Senden Sie Anfragen an KI-Dienste durch Tippen oder Spracheingabe.
2. **Apps verbinden**: Verknüpfen Sie externe Dienste oder "Apps" (z. B. Chatbots, Automatisierungen) über das Menü _Apps_.
3. **Datenschutzstatus**: Überprüfen Sie jederzeit, wie Ihre Anfrage verarbeitet wird.

## Beispiele

- Eine Chatnachricht wird per Proxy anonymisiert an ein KI-Modell weitergegeben.
- Beim Verbinden eines Drittanbieters kann superheld.app festlegen, welche Daten weitergeleitet werden.

### Alltagsmodus vs. Expertenmodus

- **Alltagsmodus**: Vorkonfiguriert, minimale Einstellungen, ideal für nicht-technische Benutzer.
- **Expertenmodus**: Detaillierte Konfiguration von Datenflüssen, Filterregeln und API-Verbindungen (siehe eigenes Kapitel).
