# Installation

Zunächst müssen Sie die App für Ihr Gerät installieren:

- **iOS & Android**: Laden Sie die App aus dem jeweiligen App Store (Apple App Store oder Google Play Store). Alternativ können Beta-Versionen über Testflight bzw. den Google Play-Beta-Kanal verteilt werden.
- **Desktop (macOS, Windows, Linux)**: Besuchen Sie die offizielle Website und laden Sie das Installationspaket herunter. Folgen Sie den Installationsanweisungen für Ihr Betriebssystem.

## Systemanforderungen

- Internetverbindung
- Für mobile Geräte: iOS 14+ / Android 8+
- Für Desktop: Aktuelle Version eines unterstützten Betriebssystems

> **Hinweis:** Die App läuft lokal und benötigt keine ständige Verbindung zu einem zentralen Server. Sie fungiert als Proxy, der bei Bedarf Anfragen weiterleitet.
