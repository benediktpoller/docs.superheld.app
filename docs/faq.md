# Häufige Fragen (FAQ)

**Frage:** Was macht superheld.app genau?

**Antwort:** Die App agiert als Proxy zu KI-Diensten und dient dazu, Ihre Anfragen zu anonymisieren und den Datenfluss zu kontrollieren.

**Frage:** Kann ich die App offline verwenden?

**Antwort:** Grundsätzlich ja; viele Funktionen laufen lokal. Für KI-Anfragen ist allerdings eine Internetverbindung nötig.

**Frage:** Ist meine Privatsphäre wirklich sicher?

**Antwort:** Das System ist darauf ausgelegt, so wenig Daten wie möglich weiterzuleiten. In sensiblen Szenarien sollten Sie den Expertenmodus nutzen und eigene Regeln definieren.

Weitere Fragen werden regelmäßig ergänzt.
