# Expertenmodus & Konfiguration

Im Expertenmodus erhalten Sie volle Kontrolle über die Funktionsweise von superheld.app. Diese Einstellungen sind für Entwickler und fortgeschrittene Anwender gedacht.

## Einstellungen

- **Datenpfade**: Whitelist / Blacklist von Domains und Diensten.
- **Anonymisierung**: Wahl zwischen verschiedenen Strategien (z. B. vollständiges Scrubbing, Tokenisierung).
- **API-Schlüssel**: Hinterlegen und verwalten von Zugangsdaten zu KI-Anbietern.

> **Achtung:** Falsche Einstellungen können dazu führen, dass Anfragen fehlschlagen oder Daten ungewollt weitergegeben werden. Nutzen Sie den Expertenmodus nur, wenn Sie sich mit Netzwerken und Datenschutz auskennen.

## Tipps für Entwickler

- Verwenden Sie die eingebauten Debug-Logs, um den Datenverkehr zu analysieren.
- Testen Sie neue Regeln in einer isolierten Umgebung, bevor Sie sie produktiv einsetzen.
