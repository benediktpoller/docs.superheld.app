## Wähle dein Betriebssystem

### 📱 iOS
1. Öffne den **App Store**
2. Suche nach "superheld"
3. Tippe auf **Installieren**

**Anforderungen:** iOS 14.0+

### 🤖 Android
1. Öffne **Google Play Store**
2. Suche nach "superheld"
3. Tippe auf **Installieren**

**Anforderungen:** Android 8.0+

### 🖥️ macOS
1. Lade herunter: [superheld.app/download](https://superheld.app/download)
2. Ziehe App in **Anwendungen**
3. Starten (Cmd+Space → "superheld")

**Anforderungen:** macOS 10.14+

### 🪟 Windows
1. Lade Installer herunter: [superheld.app/download](https://superheld.app/download)
2. Starte **superheld-setup.exe**
3. Folge dem Assistenten

**Anforderungen:** Windows 10+

### 🐧 Linux
```bash
# Ubuntu/Debian
sudo apt install superheld-app

# Fedora
sudo dnf install superheldapp

# Arch
sudo pacman -S superheld-app
```

**Anforderungen:** Linux mit glibc 2.29+
