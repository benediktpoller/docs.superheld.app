## Sicherheit beim Setup

{{< callout type="warning" >}}
**Sicheres Passwort ist essentiell!**

Verwende:
- Mindestens 12 Zeichen
- Groß- & Kleinbuchstaben
- Zahlen & Sonderzeichen
- Nicht dein Geburtsdatum/Name

Beispiel: `K7@mP9$wQ2#vR8!tL`
{{< /callout >}}

### Face ID / Touch ID aktivieren

{{< details title="Biometric Lock Setup" >}}

Diese Funktion sperrt deine App automatisch:

{{% steps %}}

### Settings → Security

Öffne die Sicherheitseinstellungen

### "Biometric Authentication" aktivieren

Wähle deine Methode: Face ID, Touch ID, Fingerprint oder PIN

### Auto-Lock konfigurieren

2 Minuten empfohlen

{{% /steps %}}

{{< callout type="info" >}}
Mit Biometric Lock + 2 Minuten Auto-Lock:
- Hacker können nicht eindringen
- Du musst nicht ständig entsperren
- Daten sind geschützt wenn Handy gestohlen wird
{{< /callout >}}

{{< /details >}}

## Troubleshooting

{{< details title="Installation fehlgeschlagen?" >}}

**Problem:** App bricht gleich ab

{{< callout type="info" >}}
1. Gerät neustarten
2. App-Cache löschen:
   - iOS: Settings → superheld.app → Delete Offscreen Files
   - Android: Settings → Apps → superheld → Storage → Clear Cache
3. App neu installieren
{{< /callout >}}

---

**Problem:** Sync funktioniert nicht

{{< callout type="info" >}}
1. Check Internet-Verbindung (Wi-Fi oder Cellular)
2. App neustarten
3. Settings → Accounts → Re-sync
4. Falls immer noch nicht: Abmelden/Anmelden
{{< /callout >}}

---

**Problem:** Passwort vergessen

{{< callout type="warning" >}}
1. Tippe "Forgot Password?"
2. Verifiziere deine Identität (Email Verification Code oder Backup Recovery Code)
3. Neues Passwort setzen

**Backup Recovery Code speichern:** Settings → Security → Save Recovery Code
{{< /callout >}}
