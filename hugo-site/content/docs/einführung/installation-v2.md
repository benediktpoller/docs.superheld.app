## 🔐 Sicherheit beim Setup

{{% alert title="⚠️ Wichtig" color="warning" %}}
**Sicheres Passwort ist essentiell!**

Verwende:
- ✅ Mindestens 12 Zeichen
- ✅ Groß- & Kleinbuchstaben
- ✅ Zahlen & Sonderzeichen
- ✅ Nicht dein Geburtsdatum/Name

Beispiel: `K7@mP9$wQ2#vR8!tL`
{{% /alert %}}

### Face ID / Touch ID aktivieren

{{% expand title="Biometric Lock Setup" %}}

Diese Funktion sperrt deine App automatisch:

```
1. Settings → Security
2. "Biometric Authentication" aktivieren
3. Wähle Methode:
   - Face ID (iPhone X+)
   - Touch ID (iPhone 5s+)
   - Fingerprint (Android)
   - PIN (alle Geräte, Fallback)
4. Auto-Lock: 2 Minuten empfohlen
```

{{% alert title="✅ Best Practice" color="success" %}}
Mit Biometric Lock + 2 Minuten Auto-Lock:
- Hacker können nicht eindringen
- Du musst nicht ständig entsperren
- Daten sind geschützt wenn Handy gestohlen wird
{{% /alert %}}

## 📱 Mehrere Geräte sync

### Geräte koppeln

{{% tabs %}}
{{% tab title="iPhone + iPad" %}}

```
Auf iPhone:
1. Settings → iCloud
2. "Sync enabled"
3. Apple ID eingeben

Auf iPad:
1. Same Apple ID Konto
2. Sync startet automatisch
3. Alle Daten sofort da ✅
```

{{% /tab %}}

{{% tab title="iPhone + Android" %}}

```
Auf iPhone:
1. Settings → Backup
2. "Manual Cloud Backup"
3. Tippe "Backup Now"

Auf Android:
1. Settings → Accounts
2. Füge gleichen Account hinzu
3. "Sync files from backup"
4. Restore startet automatisch ✅
```

{{% /tab %}}

{{% tab title="Alle Geräte" %}}

```
Universal-Methode (überall):
1. Settings → Account
2. Klick "Device Management"
3. Neue Geräte siehe "Pending Devices"
4. Bestätige auf neuem Gerät
5. Sync startet automatisch ✅
```

## 🆘 Troubleshooting

{{% expand title="Installation fehlgeschlagen?" %}}

**Problem:** App bricht gleich ab

{{% alert title="Lösung" color="info" %}}
```
1. Gerät neustarten
2. App-Cache löschen:
   - iOS: Settings → superheld.app → Delete Offscreen Files
   - Android: Settings → Apps → superheld → Storage → Clear Cache
3. App neu installieren
```
{{% /alert %}}

---

**Problem:** Sync funktioniert nicht

{{% alert title="Lösung" color="info" %}}
```
1. Check Internet-Verbindung (Wi-Fi oder Cellular)
2. App neustarten
3. Settings → Accounts → Re-sync
4. Falls immer noch nicht: Abmelden/Anmelden
```
{{% /alert %}}

---

**Problem:** Passwort vergessen

{{% alert title="Lösung" color="warning" %}}
```
1. Tippe "Forgot Password?"
2. Verifiziere deine Identität:
   - Email Verification Code, ODER
   - Backup Recovery Code
3. Neues Passwort setzen
4. Fertig ✅

💾 Backup Recovery Code speichern:
   Settings → Security → Save Recovery Code
```
{{% /alert %}}
