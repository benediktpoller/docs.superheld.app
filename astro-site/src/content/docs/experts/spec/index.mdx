Engineering-grade specifications for the Superheld platform. These documents describe **how the system works** at implementation level — components, data contracts, trust boundaries, failure modes.

:::caution[Audience]
These specifications are intended for Superheld engineers, security auditors, and integration architects. They contain implementation details that go beyond the public documentation. For integration guidance, see the [Integrator-Quickstart](/experts/integrator-quickstart).
:::

:::note[TODO-ENG Policy]
Missing engineering facts are marked with `TODO-ENG-###` identifiers. Each TODO links to either `IMPLEMENTATION_FACTS.md` or `API_FACTS.md` — structured questionnaires that the engineering team must complete to unblock these items. See the [TODO Summary](/experts/spec/todo-summary) for the full backlog.
:::
